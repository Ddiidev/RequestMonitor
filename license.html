Por que escolhemos a Licença MIT?

A Licença MIT foi escolhida para o Request Monitor pelos seguintes motivos:

  • Simplicidade: É uma das licenças mais simples e fáceis de entender
  • Permissividade: Permite máxima liberdade para os usuários
  • Compatibilidade: É compatível com a maioria das outras licenças
  • Adoção Ampla: É amplamente aceita na comunidade de desenvolvimento
  • Flexibilidade Comercial: Permite uso em projetos comerciais

Dependências e Bibliotecas

A extensão Request Monitor é desenvolvida usando apenas APIs nativas do Chrome e JavaScript vanilla, não dependendo de bibliotecas externas. Isso garante:

  • Menor complexidade de licenciamento
  • Melhor performance
  • Maior segurança
  • Facilidade de auditoria
