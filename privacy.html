Política de Privacidade

Resumo: A extensão Request Monitor processa todos os dados localmente no seu navegador. Nenhuma informação pessoal ou dados de navegação são coletados, armazenados ou transmitidos para servidores externos.

1. Informações Gerais

Esta Política de Privacidade descreve como a extensão Request Monitor ("nós", "nossa" ou "extensão") coleta, usa e protege suas informações quando você utiliza nossa extensão do Chrome.

2. Dados Coletados

2.1 Dados NÃO Coletados

A extensão Request Monitor NÃO coleta:

  • Informações pessoais identificáveis
  • Histórico de navegação
  • URLs visitadas
  • Conteúdo de páginas web
  • Dados de formulários
  • Cookies ou dados de sessão
  • Informações de localização
  • Dados de autenticação

2.2 Dados Processados Localmente

A extensão processa localmente apenas:

  • Metadados de Requisições HTTP: Status, duração e tipo de requisições (apenas para fins de monitoramento)
  • Configurações do Usuário: Duração mínima para notificações, idioma preferido e estado de ativação
  • Estado da Extensão: Quais abas estão sendo monitoradas e contadores de requisições

3. Como os Dados São Usados

3.1 Finalidade do Processamento

Os dados processados localmente são utilizados exclusivamente para:

  • Detectar quando requisições HTTP são completadas
  • Reproduzir notificações sonoras conforme configurado
  • Exibir estatísticas em tempo real na interface da extensão
  • Manter suas preferências de configuração

3.2 Armazenamento Local

As configurações da extensão são armazenadas localmente usando a API chrome.storage.sync do Chrome, que:

  • Sincroniza suas configurações entre dispositivos logados na mesma conta Google
  • Mantém os dados criptografados
  • Não permite acesso por terceiros

4. Permissões da Extensão

4.1 Justificativa das Permissões

A extensão solicita as seguintes permissões:

  • "debugger": Para interceptar requisições de rede e detectar quando são completadas
  • "activeTab": Para acessar a aba ativa e reproduzir sons de notificação
  • "storage": Para salvar suas configurações localmente
  • "tabs": Para gerenciar o monitoramento por aba
  • "notifications": Para exibir notificações do sistema (se habilitado)
  • "<all_urls>": Para funcionar em todos os sites que você visitar

4.2 Uso Responsável das Permissões

Embora a extensão tenha permissão para acessar todos os sites, ela:

  • NÃO lê o conteúdo das páginas
  • NÃO modifica páginas web
  • NÃO intercepta dados pessoais
  • Usa as permissões apenas para monitorar metadados de requisições HTTP

5. Compartilhamento de Dados

A extensão Request Monitor NÃO compartilha nenhum dado com terceiros, pois:

  • Não coleta dados pessoais
  • Não se conecta a servidores externos
  • Não possui funcionalidades de analytics ou tracking
  • Todo processamento é feito localmente no seu navegador

6. Segurança

A segurança dos seus dados é garantida através de:

  • Processamento Local: Todos os dados permanecem no seu dispositivo
  • Código Aberto: O código fonte está disponível para auditoria pública
  • Sem Conexões Externas: A extensão não faz requisições para servidores externos
  • Princípio da Menor Permissão: Usa apenas as permissões mínimas necessárias

7. Seus Direitos

Como usuário, você tem o direito de:

  • Desinstalar: Remover a extensão a qualquer momento
  • Configurar: Ajustar todas as configurações conforme sua preferência
  • Desativar: Pausar o monitoramento sem desinstalar a extensão
  • Auditar: Examinar o código fonte no repositório GitHub

8. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada ocasionalmente. Quaisquer alterações serão:

  • Publicadas nesta página
  • Indicadas pela data de "Última atualização"
  • Comunicadas através de atualizações da extensão, se necessário

9. Contato

Se você tiver dúvidas sobre esta Política de Privacidade ou sobre o funcionamento da extensão, você pode:

  • Abrir uma issue no repositório GitHub
  • Examinar o código fonte para entender como a extensão funciona
  • Consultar a documentação técnica no README

Compromisso: Estamos comprometidos em manter a transparência total sobre como a extensão funciona e em proteger sua privacidade através do processamento exclusivamente local de dados.
